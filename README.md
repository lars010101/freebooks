# freeBooks

Open-source double-entry accounting workbook for small companies.

**Stack:** DuckDB · Node.js (Express) · Evidence.dev  
**License:** AGPL-3.0

## Structure

```
db/          Schema and migration scripts
api/         Node.js Express API (ported from GCP Cloud Function)
reports/     Evidence.dev financial reports
data/        Local data files — gitignored, never committed
archive_google/  Original Google Sheets/BigQuery implementation
```
